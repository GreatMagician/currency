Клиент-серверное приложение, которое взаимодействует с сервисом CBR (http://www.cbr.ru/
 для получения курса  валют.
 Одна страница пользовательского интерфейса. На странице есть возможность задать временной период,
 тип валюты. Вывод курсов валют  в табличном виде.
 Стек: Spring Boot, jsp, maven, Jquery, json.
Для запуска проекта - spring-boot:run
